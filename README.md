# ai-name-generator

## 项目概述
基于 **Vue3 + FastAPI** 构建的开源 AI 取名全栈项目，实现多场景名字生成、查重、寓意解析功能，适合全栈开发者入门学习与二次定制开发。

## 技术栈
- 前端：Vue3 + Vite + Pinia（轻量高效，易上手、易定制）
- 后端：FastAPI + Python（异步高性能，自动生成 API 文档，便于调试与扩展）

## 核心特性
✅ 前后端分离架构，低耦合、易扩展
✅ 多风格取名场景，支持自定义新增取名规则
✅ 完整的 API 接口，便于对接其他前端或客户端
✅ 简洁美观的交互界面，开箱即用
✅ 详细的名字解析逻辑，可二次优化迭代
